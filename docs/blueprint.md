# **App Name**: Portfolio Pro

## Core Features:

- Profile Showcase: Display your resume and professional details.
- Project Portfolio: Showcase personal projects with descriptions, tech stacks, and links to live demos or repositories.
- Skills Visualization: Visualize technical skills and competencies with interactive charts or graphs.
- Achievement Highlights: Highlight key accomplishments and certifications.
- Personalized Introduction: Use generative AI to generate different introductions depending on who you want to target the portfolio at; AI will be a tool used by the user to rewrite their 'about me'.
- Contact Form: Allow visitors to send messages.

## Style Guidelines:

- Primary color: Deep blue (#34495E) to convey professionalism and trust.
- Background color: Light gray (#F4F4F8) for a clean and modern look.
- Accent color: Teal (#008080) to highlight key information and calls to action.
- Body font: 'Inter', a grotesque-style sans-serif, for clean and modern readability.
- Headline font: 'Space Grotesk' sans-serif, used for headlines and short amounts of body text. For longer text use 'Inter' instead.
- Use minimalist icons to represent skills and technologies.
- Clean, single-page layout with smooth scrolling and clear sections.
- Subtle transitions and animations to enhance user experience.
